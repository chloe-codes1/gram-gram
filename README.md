# Gram-gram

> Instagram clone with Django

<br>

<br>

## Features

<br>

- #### Post CRUD
  
  - Image upload
  
- #### Comment 
  
  - Reply on comment
  
- #### Follow
  
  - Follower/Following List
  
- #### Post likes

- #### Authentication

  - `Google Login API`
  - `Github Login API`

- #### Hash tags
  
  - Sort posts by hash tag
  - Search by hash tag
  
- #### Profile
  
  - Profile photo by using [gravatar](https://en.gravatar.com/)
